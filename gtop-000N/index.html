Definition. Radon Probability Measure [gtop-000N]

  • March 10, 2026
  • Zihang Zeng

A Radon probability measure on a locally compact Hausdorff space \(X\) is a Radon measure \(\mu \) satisfying \(\mu (X) = 1\).

The space of all Radon probability measures on \(X\) is denoted \(\mathcal {P}(X)\). When \(X\) is a compact metric space, \(\mathcal {P}(X)\) equipped with the weak-* topology is itself a compact metric space.

      Definition. Radon Measure [gtop-000A]

      • March 11, 2026
      • Zihang Zeng

      A Radon measure on a locally compact Hausdorff space \(X\) is a Borel measure \(\mu \) satisfying:

      1. \(\mu (K) < \infty \) for every compact set \(K \subseteq X\).
      2. \(\mu \) is outer regular on all Borel sets.
      3. \(\mu \) is inner regular on all open sets.

      Definition. Metric Space [gtop-0009]

      • March 10, 2026
      • Zihang Zeng

      A metric space is an ordered pair \((M,d)\) where \(M\) is a set and \(d\) is a metric on \(M\), i.e., a function \(d:M\times M \to \mathbb {R}\) satisfying the following axioms for all points \(x,y,z \in M\):

      • \(d(x,x) = 0\)
      • \(\text {If } x\neq y,\text { then } d(x,y)>0\). (Positivity)
      • \(d(x,y) = d(y,x)\). (Symmetry)
      • \(d(x,z) \leq d(x,y) + d(y,z)\). (Triangle inequality)

      Definition. Compactness [gtop-0008]

      • March 7, 2026
      • Zihang Zeng

      A topological space \((X, \tau )\) is compact if every open cover of \(X\) has a finite subcover.

      A subset \(K \subseteq X\) is compact if it is compact as a topological space with the subspace topology, or equivalently, if every open cover of \(K\) by open sets in \(X\) has a finite subcover.

      In Euclidean space \(\mathbb {R}^n\), the Heine-Borel theorem gives a concrete characterization: a subset \(K \subseteq \mathbb {R}^n\) is compact if and only if it is closed and bounded (i.e. \(K \subseteq B(0, R)\) for some \(R > 0\)).
