Optimality Conditions for Bolza Problem [opt-6004]

  • February 22, 2026
  • Zihang Zeng

The necessary conditions for the optimality of \(\mathbf {u}(t)\) are given by the following set of equations:

1. State Equation

  • February 22, 2026
  • Zihang Zeng
 \[ \dot {\mathbf {x}} = f\left ( \mathbf {x}, \mathbf {u} \right ), \quad \mathbf {x}(0) = \mathbf {x}_0 \]

2. Costate Equation

  • February 22, 2026
  • Zihang Zeng
 \[ \dot {\mathbf {p}} = -\left ( \frac {\partial f}{\partial \mathbf {x}} \right )^\intercal \mathbf {p} - \left ( \frac {\partial L}{\partial \mathbf {x}} \right )^\intercal \]

3. Stationarity Condition

  • February 22, 2026
  • Zihang Zeng
 \[ \left ( \frac {\partial f}{\partial \mathbf {u}} \right )^\intercal \mathbf {p} + \left ( \frac {\partial L}{\partial \mathbf {u}} \right )^\intercal = 0 \]

4. Boundary Condition (Transversality)

  • February 22, 2026
  • Zihang Zeng
 \[ \mathbf {p}(t_f) = \nabla \Phi \left ( \mathbf {x}(t_f) \right ) \]
