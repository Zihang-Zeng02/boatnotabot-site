Definition 1.1. Bolza Problem [opt-6002]

  • February 21, 2026
  • Zihang Zeng

Bolza problem is a fundamental formulation in the calculus of variations and optimal control that minimizes a cost functional consisting of both an integral term (running cost) and a terminal cost. It is described with a system dynamic funcation and a cost function. \[\dot {\mathbf {x}} = f(\mathbf {x}, \mathbf {u}),\quad \mathbf {x} (0) = x_0\]\[J := \int _0^{t_f} L \big (\mathbf {x}(t), \mathbf {u}(t)\big ) dt + \Phi \left ( \mathbf {x}(t_f) \right )\] where state \(\mathbf {x}(t)\in \mathbb {R}^n\), control \(\mathbf {u}(t)\in \mathbb {R}^k\), \(L:\mathbb {R}^n \times \mathbb {R}^k \to \mathbb {R}\) is the running cost and \(\Phi :\mathbb {R}^n \to \mathbb {R}\) is the terminal cost. The goal is to find the optimal control \(\mathbf {u}^*(t)\in \mathbb {R}^k\) that minimizes this cost functional subject to the system dynamics and boundary conditions. With input constraint, we minimize \(J\) subject to constraints \(\mathbf {u}(t) \in U_t \subseteq \mathbb {R} ^k\) where \(U_t\) is the set of admissible control at time \(t\) and is guaranteed to be compact and convex.

Definition 1.2. Admissible control [opt-6006]

  • March 7, 2026
  • Zihang Zeng

We say that a control \(\mathbf {u}\) is admissible if \(\forall t \in [0, t_f],\ \mathbf {u}(t) \in U_t\) where \(U_t\) is the control constraint set.

2.1. First-order Variation of the Bolza Functional [opt-6003]

  • February 22, 2026
  • Zihang Zeng

To find the optimal control \(\mathbf {u}\), we consider the first-order variation of the functional \(J\) with respect to \(\mathbf {u}\). By augmenting the cost functional with system dynamics \(\dot {\mathbf {x}} = f(\mathbf {x}, \mathbf {u})\) using the costate \(\mathbf {p}\), we define:

\[ \frac {\partial J}{\partial \mathbf {u}} = \left ( \frac {\partial f}{\partial \mathbf {u}} \right )^\intercal \mathbf {p} + \left ( \frac {\partial L}{\partial \mathbf {u}} \right )^\intercal \]

We obtain the stationarity condition \(\frac {\partial J}{\partial \mathbf {u}} \equiv 0\). See the derivation for how this expression arises from first principles.

\[\dot {\mathbf {x}}=f(\mathbf {x}, \mathbf {u}),\quad \mathbf {x}(0)=x_0\]

Definition 2.2.2. Costate Equation [opt-6009]

  • March 7, 2026
  • Zihang Zeng
 \[\dot {\mathbf {p}} = -\left ( \frac {\partial f}{\partial \mathbf {x}} \right )^\intercal \mathbf {p} - \left ( \frac {\partial L}{\partial \mathbf {x}} \right )^\intercal , \quad \mathbf {p}(t_f) = \nabla \Phi \left ( \mathbf {x}(t_f) \right )\]

Definition 2.2.3. Stationarity Condition [opt-6012]

  • March 8, 2026
  • Zihang Zeng

\[\left (\frac {\partial f}{\partial \mathbf {u}}\right )^\intercal \mathbf {p} + \left (\frac {\partial L}{\partial \mathbf {u}}\right )^\intercal \equiv 0\]

Define the Hamiltonian function at an admissible control \(\mathbf {u}\) by \[H(\mathbf {x}, \mathbf {u}, \mathbf {p}) = {\mathbf {p}}^ \intercal f(\mathbf {x},\mathbf {u}) + L(\mathbf {x}, \mathbf {u})\]

Theorem 3.2. Pontryagin's Maximum Principle [opt-6008]

  • March 7, 2026
  • Zihang Zeng

Given a control signal \(\mathbf {u}\), denote by \(\mathbf {x}_u\) and \(\mathbf {p}_u\) the corresponding state and costate trajectories.

Suppose that \(\mathbf {u}\) is an admissible optimal control. Then \(\forall t \in [0, t_f],\forall \mathbf {v} \in U_t\), \[H\big (\mathbf {x}_u(t),\mathbf {u}(t),\mathbf {p}_u(t)\big ) \leq H\big (\mathbf {x}_u(t), \mathbf {v}, \mathbf {p}_u(t)\big )\]

4. Relaxed Control

Definition. Relaxed Control [opt-6013]

  • March 10, 2026
  • Zihang Zeng

Consider the bolza problem subject to \(\dot {\mathbf {x}} (t) = f(t,\mathbf {x} (t), \mathbf {u} (t)),\ \mathbf {u} (t) \in U\), where \(U\) is a compact metric space.

Relaxed controls replace the set \(U\) by the space of Radon probability measures on \(U\).
