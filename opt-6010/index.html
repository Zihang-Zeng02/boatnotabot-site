Derivation of the First-order Variation [opt-6010]

  • March 7, 2026
  • Zihang Zeng

We derive \(\frac {\partial J}{\partial \mathbf {u}}\) for the Bolza problem from first principles, then show how the costate arises naturally.

1. Linearization of \(J\)

  • March 7, 2026
  • Zihang Zeng

Expanding \(J(\mathbf {u} + \Delta \mathbf {u}) - J(\mathbf {u})\) to first order:

\[ \int _0^{t_f} \frac {\partial L}{\partial \mathbf {x}} \Delta \mathbf {x} \, dt + \int _0^{t_f} \frac {\partial L}{\partial \mathbf {u}} \Delta \mathbf {u} \, dt + \frac {d\Phi }{d\mathbf {x}}\left ( \mathbf {x}(t_f) \right ) \Delta \mathbf {x}(t_f) \]

2. Variation of the state

  • March 7, 2026
  • Zihang Zeng

Linearizing the dynamics \(\dot {\mathbf {x}} = f(\mathbf {x}, \mathbf {u})\) gives

\[\dot {\Delta \mathbf {x}}(t) = \frac {\partial f}{\partial \mathbf {x}} \Delta \mathbf {x}(t) + \frac {\partial f}{\partial \mathbf {u}} \Delta \mathbf {u}(t)\]

whose solution, with \(\Delta \mathbf {x}(0) = 0\), is

\[\Delta \mathbf {x}(t) = \int _0^{t} \Psi (t, \tau ) \frac {\partial f}{\partial \mathbf {u}}\left ( \mathbf {x}(\tau ), \mathbf {u}(\tau ) \right ) \Delta \mathbf {u}(\tau ) \, d\tau \]

where \(\Psi (t, \tau )\) is the state transition matrix of \(\frac {\partial f}{\partial \mathbf {x}}\).

3. Collecting \(\Delta \mathbf {u}\) terms

  • March 7, 2026
  • Zihang Zeng

Substituting \(\Delta \mathbf {x}\) and swapping integration order (Fubini), all three terms become inner products against \(\Delta \mathbf {u}(\tau )\), yielding

\[ \left ( \frac {\partial J}{\partial \mathbf {u}}(\tau ) \right )^\intercal = \underbrace {\left ( \int _\tau ^{t_f} \frac {\partial L}{\partial \mathbf {x}} \Psi (t, \tau ) \, dt + \frac {d\Phi }{d\mathbf {x}}\left ( \mathbf {x}(t_f) \right ) \Psi (t_f, \tau ) \right )}_{\mathbf {p}^\intercal (\tau )} \frac {\partial f}{\partial \mathbf {u}}\left ( \mathbf {x}(\tau ), \mathbf {u}(\tau ) \right ) + \left ( \frac {\partial L}{\partial \mathbf {u}}\left ( \mathbf {x}(\tau ), \mathbf {u}(\tau ) \right ) \right )^\intercal \]

The bracketed expression is precisely the costate \(\mathbf {p}(\tau )\), so we recover

\[\frac {\partial J}{\partial \mathbf {u}} = \left ( \frac {\partial f}{\partial \mathbf {u}} \right )^\intercal \mathbf {p} + \left ( \frac {\partial L}{\partial \mathbf {u}} \right )^\intercal \]

    First-order Variation of the Bolza Functional [opt-6003]

    • February 22, 2026
    • Zihang Zeng

    To find the optimal control \(\mathbf {u}\), we consider the first-order variation of the functional \(J\) with respect to \(\mathbf {u}\). By augmenting the cost functional with system dynamics \(\dot {\mathbf {x}} = f(\mathbf {x}, \mathbf {u})\) using the costate \(\mathbf {p}\), we define:

    \[ \frac {\partial J}{\partial \mathbf {u}} = \left ( \frac {\partial f}{\partial \mathbf {u}} \right )^\intercal \mathbf {p} + \left ( \frac {\partial L}{\partial \mathbf {u}} \right )^\intercal \]

    We obtain the stationarity condition \(\frac {\partial J}{\partial \mathbf {u}} \equiv 0\). See the derivation for how this expression arises from first principles.

      Definition. Bolza Problem [opt-6002]

      • February 21, 2026
      • Zihang Zeng

      Bolza problem is a fundamental formulation in the calculus of variations and optimal control that minimizes a cost functional consisting of both an integral term (running cost) and a terminal cost. It is described with a system dynamic funcation and a cost function. \[\dot {\mathbf {x}} = f(\mathbf {x}, \mathbf {u}),\quad \mathbf {x} (0) = x_0\]\[J := \int _0^{t_f} L \big (\mathbf {x}(t), \mathbf {u}(t)\big ) dt + \Phi \left ( \mathbf {x}(t_f) \right )\] where state \(\mathbf {x}(t)\in \mathbb {R}^n\), control \(\mathbf {u}(t)\in \mathbb {R}^k\), \(L:\mathbb {R}^n \times \mathbb {R}^k \to \mathbb {R}\) is the running cost and \(\Phi :\mathbb {R}^n \to \mathbb {R}\) is the terminal cost. The goal is to find the optimal control \(\mathbf {u}^*(t)\in \mathbb {R}^k\) that minimizes this cost functional subject to the system dynamics and boundary conditions. With input constraint, we minimize \(J\) subject to constraints \(\mathbf {u}(t) \in U_t \subseteq \mathbb {R} ^k\) where \(U_t\) is the set of admissible control at time \(t\) and is guaranteed to be compact and convex.
