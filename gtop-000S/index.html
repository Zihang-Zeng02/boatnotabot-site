Definition. Open Ball [gtop-000S]

  • March 11, 2026
  • Zihang Zeng

Let \((X, d)\) be a metric space. The open ball centered at \(x \in X\) with radius \(\epsilon > 0\) is

\[B(x, \epsilon ) = \left \{ y \in X : d(x, y) < \epsilon \right \}\]

Context

    Definition. Metric Space [gtop-0003]

    • March 7, 2026
    • Zihang Zeng

    A metric space is a pair \((X, d)\) where \(X\) is a set and \(d : X \times X \to \mathbb {R}\) is a function called a metric, satisfying for all \(x, y, z \in X\):

    1. \(d(x, y) \geq 0\) and \(d(x, y) = 0\) if and only if \(x = y\) (positive definiteness).
    2. \(d(x, y) = d(y, x)\) (symmetry).
    3. \(d(x, z) \leq d(x, y) + d(y, z)\) (triangle inequality).

    Every metric space \((X, d)\) induces a topology on \(X\), called the metric topology, where the open sets are unions of open balls.

    Notation. Metric Topology [gtop-000R]

    • March 11, 2026
    • Zihang Zeng

    A metric space \((X, d)\) induces a topology \(\tau _d\) on \(X\) by declaring \(U \in \tau _d\) if and only if for every \(x \in U\) there exists \(\epsilon > 0\) such that the open ball \(B(x, \epsilon ) \subseteq U\).

    That is, a metric \(d\) induces a topology through the following chain:

    1. The metric \(d\) defines open balls \(B(x, \epsilon )\).
    2. Open balls generate open sets: a set is open if and only if it is an arbitrary union of open balls.
    3. The collection of all open sets forms \(\tau _d\), a valid topology.

    Every metric space is a topological space, but not every topological space arises from a metric. A topological space \((X, \tau )\) is called metrizable if there exists a metric \(d\) on \(X\) such that \(\tau = \tau _d\).

      Intuition. Intuition: Measure vs Metric [todo-0003]

      • March 11, 2026
      • Zihang Zeng

      A measure assigns a "size" to a set — how much stuff is in it. Length, area, volume, and probability are all measures.

      A metric assigns a "distance" between two points — how far apart they are.

      They answer fundamentally different questions: measure asks "how big is this region?" while metric asks "how far apart are these two points?"

      A metric is upstream of measure: a metric defines open balls, which give a topology, which give a Borel \(\sigma \)-algebra, which gives sets you can measure. A set can be small in measure but wide in metric (e.g., a thin line segment across a room).

      Definition. Euclidean Space [gtop-0004]

      • March 7, 2026
      • Zihang Zeng

      The Euclidean space \(\mathbb {R}^n\) is the metric space equipped with the Euclidean metric

      \[d(x, y) = \lVert x - y \rVert = \sqrt {\sum _{i=1}^{n} (x_i - y_i)^2}\]

      The topology induced by this metric is called the standard topology on \(\mathbb {R}^n\). In this topology, the open ball centered at \(x\) with radius \(\epsilon > 0\) is

      \[B(x, \epsilon ) = \left \{ y \in \mathbb {R}^n : \lVert y - x \rVert < \epsilon \right \}\]

      This is a special case of a metric space, which in turn induces a topological space.

      Definition. Open Set [gtop-0005]

      • March 7, 2026
      • Zihang Zeng

      Let \((X, \tau )\) be a topological space. A subset \(U \subseteq X\) is called open if \(U \in \tau \).

      In a Euclidean space \(\mathbb {R}^n\), a subset \(U \subseteq \mathbb {R}^n\) is open if for every \(x \in U\), there exists \(\epsilon > 0\) such that \(B(x, \epsilon ) \subseteq U\). More generally, in any metric space \((X, d)\), a set is open if it is a union of open balls.

      The Euclidean definition is a special case where the topology is induced by the metric \(d(x, y) = \lVert x - y \rVert \).
