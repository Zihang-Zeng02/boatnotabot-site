Definition. Borel Set [gtop-000L]

  • March 10, 2026
  • Zihang Zeng

A Borel set in a topological space \(X\) is any set belonging to the Borel \(\sigma \)-algebra \(\mathcal {B}(X)\).

    Intuition. Intuition: Borel Measure [todo-0006]

    • March 11, 2026
    • Zihang Zeng

    A Borel measure assigns "sizes" (lengths, areas, volumes, probabilities) to sets built from open sets.

    The topology tells you what is open, the Borel \(\sigma \)-algebra tells you what is measurable, and a Borel measure tells you how big. It is a function that assigns a non-negative size to each Borel set, respecting the rule that the measure of a countable disjoint union equals the sum of the measures.

    Why not measure all subsets? In spaces like \(\mathbb {R}\), there exist pathological sets (via the axiom of choice) that cannot be assigned a size consistently. The Borel \(\sigma \)-algebra avoids this by only including sets constructible from open sets through countable operations — which turns out to be rich enough for essentially all practical purposes.

    Definition. Radon Measure [gtop-000A]

    • March 11, 2026
    • Zihang Zeng

    A Radon measure on a locally compact Hausdorff space \(X\) is a Borel measure \(\mu \) satisfying:

    1. \(\mu (K) < \infty \) for every compact set \(K \subseteq X\).
    2. \(\mu \) is outer regular on all Borel sets.
    3. \(\mu \) is inner regular on all open sets.

    Definition. Inner Regularity [gtop-000E]

    • March 10, 2026
    • Zihang Zeng

    A Borel measure \(\mu \) on a topological space \(X\) is inner regular on a Borel set \(E\) if \[ \mu (E) = \sup \left \{ \mu (M) : M \subseteq E, \; M \text { is compact} \right \} \] where \(M\) is a compact subset.

    That is, the measure of \(E\) can be approximated from below by the measures of compact subsets of \(E\).

      Definition. Borel \(\sigma \)-algebra [gtop-000K]

      • March 10, 2026
      • Zihang Zeng

      Let \((X, \tau )\) be a topological space. The Borel \(\sigma \)-algebra \(\mathcal {B}(X)\) is the smallest \(\sigma \)-algebra on \(X\) containing all open sets.

      Since \(\mathcal {B}(X)\) is generated by the open sets, it also contains all closed sets, countable unions of closed sets (\(F_\sigma \) sets), countable intersections of open sets (\(G_\delta \) sets), and so on.

      Definition. Topological Space [gtop-0002]

      • March 7, 2026
      • Zihang Zeng

      A topological space is a pair \((X, \tau )\) where \(X\) is a set and \(\tau \subseteq \mathcal {P}(X)\) is a collection of subsets of \(X\), called a topology on \(X\), satisfying:

      1. \(\varnothing , X \in \tau \)
      2. If \(\left \{ U_\alpha \right \}_{\alpha \in A} \subseteq \tau \), then \(\bigcup _{\alpha \in A} U_\alpha \in \tau \) (closed under arbitrary unions).
      3. If \(U_1, U_2 \in \tau \), then \(U_1 \cap U_2 \in \tau \) (closed under finite intersections).

      The elements of \(\tau \) are called open sets. A set \(C \subseteq X\) is called closed if its complement \(X \setminus C\) is open.

      Example. Topologies on a Three-Point Set [gtop-000P]

      • March 11, 2026
      • Zihang Zeng

      Let \(X = \left \{ 1, 2, 3 \right \}\). The following are all valid topologies on \(X\):

      • Indiscrete: \(\tau _1 = \left \{ \varnothing , X \right \}\). Only \(\varnothing \) and \(X\) are open.
      • Partial: \(\tau _2 = \left \{ \varnothing , \left \{ 1 \right \}, X \right \}\). Only \(\left \{ 1 \right \}\) and the trivial sets are open.
      • Discrete: \(\tau _3 = \mathcal {P}(X)\). Every subset is open.

      Each satisfies the three axioms of a topological space. The same set \(X\) carries different topological structure depending on the choice of \(\tau \).

      Example. Open, Closed, Both, or Neither [gtop-000Q]

      • March 11, 2026
      • Zihang Zeng

      Let \(X = \left \{ 1,2,3 \right \}\) with \(\tau = \left \{ \varnothing , \left \{ 1 \right \}, X \right \}\). A subset need not be open or closed — it can be both, or neither.

      • \(\left \{ 1 \right \}\) is open (in \(\tau \)), but not closed (\(X \setminus \left \{ 1 \right \} = \left \{ 2,3 \right \} \notin \tau \)).
      • \(\left \{ 2,3 \right \}\) is closed (\(X \setminus \left \{ 2,3 \right \} = \left \{ 1 \right \} \in \tau \)), but not open.
      • \(\varnothing \) and \(X\) are both open and closed.
      • \(\left \{ 1,2 \right \}\) is neither open (not in \(\tau \)) nor closed (\(X \setminus \left \{ 1,2 \right \} = \left \{ 3 \right \} \notin \tau \)).
