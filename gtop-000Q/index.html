Example. Open, Closed, Both, or Neither [gtop-000Q]

  • March 11, 2026
  • Zihang Zeng

Let \(X = \left \{ 1,2,3 \right \}\) with \(\tau = \left \{ \varnothing , \left \{ 1 \right \}, X \right \}\). A subset need not be open or closed — it can be both, or neither.

  • \(\left \{ 1 \right \}\) is open (in \(\tau \)), but not closed (\(X \setminus \left \{ 1 \right \} = \left \{ 2,3 \right \} \notin \tau \)).
  • \(\left \{ 2,3 \right \}\) is closed (\(X \setminus \left \{ 2,3 \right \} = \left \{ 1 \right \} \in \tau \)), but not open.
  • \(\varnothing \) and \(X\) are both open and closed.
  • \(\left \{ 1,2 \right \}\) is neither open (not in \(\tau \)) nor closed (\(X \setminus \left \{ 1,2 \right \} = \left \{ 3 \right \} \notin \tau \)).

    Definition. Topological Space [gtop-0002]

    • March 7, 2026
    • Zihang Zeng

    A topological space is a pair \((X, \tau )\) where \(X\) is a set and \(\tau \subseteq \mathcal {P}(X)\) is a collection of subsets of \(X\), called a topology on \(X\), satisfying:

    1. \(\varnothing , X \in \tau \)
    2. If \(\left \{ U_\alpha \right \}_{\alpha \in A} \subseteq \tau \), then \(\bigcup _{\alpha \in A} U_\alpha \in \tau \) (closed under arbitrary unions).
    3. If \(U_1, U_2 \in \tau \), then \(U_1 \cap U_2 \in \tau \) (closed under finite intersections).

    The elements of \(\tau \) are called open sets. A set \(C \subseteq X\) is called closed if its complement \(X \setminus C\) is open.

    Example. Topologies on a Three-Point Set [gtop-000P]

    • March 11, 2026
    • Zihang Zeng

    Let \(X = \left \{ 1, 2, 3 \right \}\). The following are all valid topologies on \(X\):

    • Indiscrete: \(\tau _1 = \left \{ \varnothing , X \right \}\). Only \(\varnothing \) and \(X\) are open.
    • Partial: \(\tau _2 = \left \{ \varnothing , \left \{ 1 \right \}, X \right \}\). Only \(\left \{ 1 \right \}\) and the trivial sets are open.
    • Discrete: \(\tau _3 = \mathcal {P}(X)\). Every subset is open.

    Each satisfies the three axioms of a topological space. The same set \(X\) carries different topological structure depending on the choice of \(\tau \).

      Definition. Closed Set [gtop-0006]

      • March 7, 2026
      • Zihang Zeng

      Let \((X, \tau )\) be a topological space. A subset \(C \subseteq X\) is called closed if its complement \(X \setminus C\) is open, i.e. \(X \setminus C \in \tau \).

      Equivalently, the collection of closed sets satisfies:

      1. \(\varnothing \) and \(X\) are closed.
      2. Arbitrary intersections of closed sets are closed.
      3. Finite unions of closed sets are closed.

      Theorem. Closed Set Characterization in Metric Spaces [gtop-000U]

      • March 11, 2026
      • Zihang Zeng

      Let \((X, d)\) be a metric space and \(C \subseteq X\). Then \(C\) is closed if and only if \(C\) contains all its limit points.

      Equivalently, \(C\) is closed if and only if for every convergent sequence \(\left \{ x_n \right \} \subseteq C\) with \(x_n \to x\), we have \(x \in C\).

      This characterization relies on the metric structure. In a general topological space, sequences may not suffice to detect all limit points.

      Definition. Open Set [gtop-0005]

      • March 7, 2026
      • Zihang Zeng

      Let \((X, \tau )\) be a topological space. A subset \(U \subseteq X\) is called open if \(U \in \tau \).

      In a Euclidean space \(\mathbb {R}^n\), a subset \(U \subseteq \mathbb {R}^n\) is open if for every \(x \in U\), there exists \(\epsilon > 0\) such that \(B(x, \epsilon ) \subseteq U\). More generally, in any metric space \((X, d)\), a set is open if it is a union of open balls.

      The Euclidean definition is a special case where the topology is induced by the metric \(d(x, y) = \lVert x - y \rVert \).
